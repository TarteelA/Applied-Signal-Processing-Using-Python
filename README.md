# Applied-Signal-Processing-Using-Python
This repository demonstrates signal processing techniques applied to real-world audio signals, including heartbeats and birdsongs. You'll learn to analyze and visualize signals using:

FFT: Convert time-domain signals into frequency components for quick spectral analysis.
STFT: Capture how signal frequencies change over time using short windows.
CWT: Perform detailed time-frequency analysis with wavelets for deeper insights.
Explore the power of signal decomposition and gain hands-on experience with Python code to process audio data!
